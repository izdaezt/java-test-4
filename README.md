# InmobiJavaTest4

Bài test cho vị trị Back-end

## Cài đặt môi trường

- Database: postgres
- Java SDK 17/21
- Visual Studio Code
- Nodejs

## Cách Run Project

npm start => run port riêng 9000 và gọi về api 8080
npm run webapp:build => cùng vs backend

```
./npmw install
```

```bash
./mvnw generate-sources
```
